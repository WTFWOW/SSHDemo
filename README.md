# SSHDemo
a simple Demo which contains Maven,Spring,SpringMVC,Hibernate,Bootstrap</br>
一个用Maven管理的Spring+SpringMVC+Hibernate项目</br>
项目非常简单,实现了增删改查,分页功能</br>
前端运用Bootstrap框架</br>
编码规范,结构清晰,适合初学者学习</br>
